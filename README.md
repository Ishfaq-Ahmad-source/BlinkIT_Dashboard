**Blinkit Sales Analytics Dashboard (Power BI)**
1. Overview
This project features an interactive Power BI dashboard designed to analyze sales and operational performance for a quick-commerce retail model inspired by Blinkit. It highlights key business metrics and delivers actionable insights through clear, well-structured visualizations.

2. Key Insights Delivered
Total and average sales performance
Outlet analysis by size, type, and location tier
Product category contribution to revenue
Customer satisfaction trends using average ratings
Sales impact of item visibility and fat content

3. Skills Demonstrated
Data cleaning and transformation
Data modeling and DAX calculations
Business intelligence reporting
Interactive dashboard design
Insight-driven visualization

4. Tools Used
Microsoft Power BI DAX
Data Visualization & BI Concepts
DAX

6. Purpose
Developed for academic learning and professional portfolio demonstration, showcasing practical BI and data analytics capabilities relevant to analyst and reporting roles.
